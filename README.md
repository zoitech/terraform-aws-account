# AWS generic account setup
Terraform module which setup the most generic aws-accounts settings.
* Set account alias
* Set password policy
* Add a logging bucket
* Enable CloudTail
* Create IAM_Policy


## Usage
```hcl
module "account" {
  source = "zoitech/account/aws"
  account_name = "my-aws-account"
}
```
